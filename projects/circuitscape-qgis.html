CircuitscapeQGIS (QGIS plugin) — WIP

Run Circuitscape connectivity analysis directly from QGIS. This plugin helps run circuit-theory based connectivity models using resistance rasters and focal points, with background task support and automatic CRS handling.

Status: WIP — active development. See repository link below.

Overview

CircuitscapeQGIS is a QGIS plugin that wraps Circuitscape functionality and provides a user-friendly interface to run connectivity analyses from within QGIS. The plugin supports resistance rasters, focal points, multiple solver options, and background processing via the QGIS task manager.

Features

  • Run Circuitscape directly from QGIS (resistance or conductance modes)
  • Background processing with the QGIS Task Manager
  • Automatic CRS transformation for focal points
  • Support for multiple solver options (fast AMG, precise CHOLMOD)
  • Clean, user-friendly interface

Installation

Requirements: QGIS 3.16+ and Circuitscape installed (Julia Circuitscape or Python Circuitscape).

Option 1 — Julia + Circuitscape.jl (recommended)

julia -e 'using Pkg; Pkg.add("Circuitscape")'

Option 2 — Python Circuitscape

pip install circuitscape

Usage

  1. Load a resistance raster into QGIS (higher values = harder to move through).
  2. Optionally load focal points.
  3. Open the CircuitscapeQGIS plugin, configure inputs, choose a solver and output location.
  4. Run and monitor progress in the QGIS Task Manager. Output is added to the map when ready.

Troubleshooting & Tips

  • Ensure Circuitscape or Julia is in your PATH.
  • Use a projected CRS (not lat/long) for accurate distance-based analysis.
  • Reduce raster resolution for large analyses to save time.
